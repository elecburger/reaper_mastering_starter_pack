Welcome to the Mastering Explained REAPER Starter Pack!

This package contains a pre-made configuration with additional scripts for the REAPER DAW that will help you get started mastering. 

It is used in the course "Audio Mastering Fundamentals" from Mastering Explained.

See https://www.masteringexplained.com/mastering-course for more details!
